🚀 What is ElectraMind?
ElectraMind is a full-stack, interactive platform that lets you simulate real elections using different voting systems — and instantly see how outcomes change.

Because here’s the truth:

The way we count votes can matter more than the votes themselves.

Instead of reading abstract theory, ElectraMind lets you:

Run elections
Compare systems
Visualize results
And experience the impact of voting rules
⚡ Why This Project Exists
Most people think elections are simple: 👉 “More votes = winner”

But reality is very different.

The same group of voters can produce:

Completely different winners
Different levels of fairness
Different representation
👉 Just by changing the voting system.

ElectraMind makes that invisible truth visible, interactive, and undeniable.
